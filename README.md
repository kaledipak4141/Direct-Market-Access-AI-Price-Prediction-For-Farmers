# Direct Market Access & AI Price Prediction For Farmers

## Project Description

This project is an AI-powered agricultural marketplace that enables farmers to connect directly with buyers while providing crop price predictions using Machine Learning. The system helps farmers make informed selling decisions and improve profitability.

## Features

* Farmer Registration and Login
* Buyer Registration and Login
* Farmer Dashboard
* Buyer Dashboard
* Direct Farmer-to-Buyer Communication
* AI-Based Crop Price Prediction
* location service(Google Map)
* Crop Price Dataset Analysis
* Chat System
* MySQL Database Integration

## Installation Steps

1. Clone the repository
2. Install Python dependencies
3. Import database.sql into MySQL
4. Run train_model.py (if retraining is required)
5. Run app.py
6. Open the application in your browser
